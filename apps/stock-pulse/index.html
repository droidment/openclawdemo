> project boundaries

How this stays isolated

Everyone can share the same Firebase project as long as each app gets its own path namespace and security rules are written with those namespaces in mind.

  • Hosting route: /apps/stock-pulse/
  • Realtime path: apps/stock-pulse/watchlist
  • Firestore collection: apps_stock_pulse_watchlist
  • Data Connect table group: stock_pulse_*
